
Ocean View Resort Management System
Lecturer Ready Submission

Tech Stack
- Java JDK 17
- Apache Tomcat 9
- Maven WAR
- JSP + Servlets (MVC Architecture)
- DAO Pattern
- MySQL

Modules
- Login / Logout with session
- Dashboard
- Reservation Management (Create / View)
- Staff Management
- Billing calculation by nights
- Help page

Run Instructions
1. Create database using database.sql
2. Import project as Existing Maven Project in Eclipse
3. Run on Tomcat 9
4. Open http://localhost:8080/ocean-view-resort

Default Login
username: admin
password: admin
